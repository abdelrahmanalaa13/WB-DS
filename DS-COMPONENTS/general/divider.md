# Divider

Visual separator between content sections.

- Layer: L1 (primitive)
- Dependencies: none
- Status: done
- Baseline: ds-library.html

## Variants

- Orientation: horizontal, vertical
- Weight: default (subtle), strong
- Label: none, with centered label text

## Valid Tokens

`--border-subtle` (default weight), `--border-default` (strong weight), `--content-tertiary` (label text), `--font-size-xs` (label), `--space-3` (gap around label)

## Forbidden

- Never use Divider as a decorative border on a component. Divider is a standalone content separator.

10. Verdict

Rubik: Approved for DS v4

Rubik has native Arabic support with all 7 weights (300-900) including italics. The Arabic was professionally designed by Omaima Dajani using Naskh calligraphic style. It renders well at all sizes including 11px caption text. The slightly rounded corners give it a friendly, modern feel that matches the "Quiet Intelligence" aesthetic without being childish. Tabular numerals work correctly.

Mono Font Recommendation

JetBrains Mono RECOMMENDED Taller x-height matches Rubik's proportions. Modern, designed for code readability. Optional ligatures. 400-700 weights.
IBM Plex Mono STRONG ALT More corporate/neutral tone. Excellent Unicode. Slightly smaller x-height. Good if you want mono to feel more subdued.
Fira Code GOOD Wider characters may cause layout issues in tight ERP tables. Strong ligatures add personality but can confuse non-developers.
Source Code Pro GOOD Traditional feel, slightly narrower. Pairs better with humanist fonts than geometric ones like Rubik.

Final Font Stack

--font-sans: 'Rubik', system-ui, -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', 'Courier New', monospace;
Rubik handles both Arabic and Latin natively in a single font file (variable font). No need for a separate Arabic font. JetBrains Mono provides modern, readable code display with x-height proportions that harmonize with Rubik's geometry.
